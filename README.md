# Student Developer Portfolio App

## Project Description
This is a simple PHP and MySQL microservice-style web application built as a group project. The application allows a student developer to create a profile and manage projects through create, read, update, and delete operations.

## Group Members and Roles
- Member 1: Frontend/UI
- Member 2: Profile Service
- Member 3: Project Create/View Service
- Member 4: Project Update/Delete Service
- Member 5: Database and Integration
- Member 6: Testing and Documentation

## Features
- Create profile
- Update profile
- Add project
- View projects
- Edit project
- Delete project

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- GitHub

## Project Structure
- config/
- database/
- frontend/
- profile-service/
- project-service/

## Setup Steps
1. Create the database tables using `database/schema.sql`
2. Update `config/db.php` with the correct database credentials
3. Upload the project files to your hosting environment
4. Open the application in the browser
